# ## mv-lst-d
Preset: move the most recently modified file from `~/Downloads` to the current
directory.


### mv-lst-pic
Preset: move the latest screenshot from `~/Pictures/Screenshots/` into the
current directory as `image.png`.

```sh
mv-lst-pic
```
